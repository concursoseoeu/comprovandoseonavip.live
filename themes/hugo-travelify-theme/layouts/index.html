Este ComprovandoSEOnaVIP é leve

Site estático. Sem wordpress. Sem plug-ins. Poucos recursos além do conteúdo que só será lido pelos concorrentes, ou talvez todos eles apenas vão ler os títulos e dar uma olhada geral na página para ver o que fizemos aqui.

A velocidade é um fator importante, então a otimização desse aspecto está sendo feita com cuidado.

Para garantir que o site seja leve, as imagens são otimizadas para ter o menor tamanho em bytes e sem perder qualidade e agregar valor ao conteúdo.

O Java Script e CSS estão compactados para ter menor tamanho, ainda assim há possibilidade de ganhos nesse aspecto.

Para ser rápido na montagem do site, foi necessário escolher um tema que carrega alguns códigos que não serão usados. Parte deles já foi removida no CSS e no JS, mas ainda há trabalho nisso.

É um tipo de otimização que dá mais trabalho, mas trás ganhos de performance tanto em sites estáticos como em sites dinâmicos que usam wordpress e outros CMS’s.

Como não tem plug-ins no hugo, algumas coisas oferecem um desafio maior e é necessário fazer alguma pesquisa para encontrar a melhor forma, como tivemos que fazer para otimizar o vídeo que está publicado no final desse artigo.
